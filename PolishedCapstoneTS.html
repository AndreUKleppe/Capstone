Project: Accidents Time Series Forecasting Using ARIMA¶

This notebook demonstrates the analysis and forecasting of car accident data. It focuses on cleaning, analyzing, and modeling the data to extract actionable insights and predict accident trends over time. This information can assist decision-makers in understanding high-risk periods and conditions to reduce accident severity and frequency.

Time Series Forecasting with a Sampled Dataset¶

This notebook uses 1000 randomly sampled instances from the dataset to forecast periods with higher accident frequencies using ARIMA.

# Import required libraries
from statsmodels.tsa.arima.model import ARIMA
from sklearn.metrics import mean_absolute_error
import itertools

# Ensure the date index has a frequency
daily_accidents = daily_accidents.asfreq('D').fillna(0)

# Split data into training and testing sets
train_size = int(len(daily_accidents) * 0.8)
train, test = daily_accidents[:train_size], daily_accidents[train_size:]

# Define the range of parameters to search
p_values = range(0, 6)  # Autoregressive terms
d_values = range(0, 3)  # Differencing terms
q_values = range(0, 6)  # Moving average terms

# Store results
results = []

# Perform grid search
for p, d, q in itertools.product(p_values, d_values, q_values):
    try:
        # Fit ARIMA model
        model = ARIMA(train['Accident_Count'], order=(p, d, q))
        arima_results = model.fit()
        
        # Forecast on the test set
        forecast = arima_results.predict(start=test.index[0], end=test.index[-1], dynamic=False)
        
        # Calculate MAE
        mae = mean_absolute_error(test['Accident_Count'], forecast)
        
        # Store the results
        results.append({'order': (p, d, q), 'mae': mae, 'aic': arima_results.aic})
        
    except Exception as e:
        # Skip parameters that fail
        continue

# Convert results to DataFrame
results_df = pd.DataFrame(results)

# Sort by MAE (or AIC if desired)
best_model = results_df.sort_values(by='mae').iloc[0]

print(f"Best model order: {best_model['order']}")
print(f"Best model MAE: {best_model['mae']}")
print(f"Best model AIC: {best_model['aic']}")
